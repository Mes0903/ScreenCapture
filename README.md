# Intro

a toy that helps create Picture-in-picture views for any window, only support Windows now, writing by C++ and ImGui.

# How to Build

```bash
git clone https://github.com/Mes0903/ScreenCapture.git
cd ScreenCapture
mkdir build && cd build
cmake ..
cmake --build .
```
